# README

## アプリ名
観察日記

## アプリの概要
このアプリは、あなたがたいせつに育てているものを記録していくためのサービスです。  
現在は開発途中ですが、アルバムとして画像や日記を残したり、これからやることを書き記すことができます。

## アプリを作った理由
最近植物を育てているのですが、その生長過程を記録したいと思ったのがきっかけです。  
また、生長記録として以外にも、作業用メモとしても活用できたら、と考え作成しました。  
植物に限らず、なにかを育てている人のために作りました。その道の初心者にやさしいアプリを目指しています。

## 現在の主な機能
- ユーザー管理  
  新規アカウント登録、アカウント削除、ログイン、ログアウト
- 投稿（タイトル、内容）管理  
  投稿の一覧表示、新規作成、編集、削除

## 技術スタック
- バックエンド：Ruby 3.3.6 , Ruby on Rails 7.2
- データベース：PostgreSQL / Neon
- 開発環境：Docker
- インフラ・デプロイ：Render
- バージョン管理：Git / GitHub
- その他ライブラリ・ツール：Tailwind CSS

## 使い方
- まず、ログインボタンから新規登録ページにアクセスしていただき、ユーザー登録を行っていただきます。
- ログインすると、投稿を作成できるようになります。作成した投稿は、後から編集及び削除できます。
- 投稿等が終わりましたら、ログアウトボタンからログアウトできます。
- ご不要になりましたら、アカウント削除ボタンからデータを削除できます。
